# CodeTrail Landing Page

Landing page oficial do CodeTrail, separada do app mobile e focada em aquisicao para a experiencia web.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- lucide-react

## O que esta nesta entrega

- hero de conversao para atrair usuarios
- narrativa do produto
- secoes de funcionalidades
- destaque para divulgacao da versao mobile
- roadmap em timeline
- CTA final pronto para publicacao

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Validacao

```bash
npm run lint
npm run build
```

## Deploy na Vercel

1. importe a pasta `CodeTrail-LandingPage` na Vercel
2. mantenha o framework como `Next.js`
3. use o comando padrao de build `npm run build`
4. use o output padrao do Next
5. publique
